# Happy Garten - Facility Services Website

This is a Next.js project for Happy Garten, a facility services company in Hamburg.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Forms:** React Hook Form + Zod
- **Language:** TypeScript

## Project Structure

- `src/app`: App Router pages and API routes.
- `src/components`: Reusable UI components (Atomic Design: atoms, molecules, organisms, templates).
- `src/lib`: Utility functions and types.
- `public`: Static assets (images).

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Build for production:**

    ```bash
    npm run build
    ```

4.  **Start production server:**

    ```bash
    npm start
    ```

## Features

- **Responsive Design:** Fully responsive layout for mobile, tablet, and desktop.
- **SEO Optimized:** Metadata configured for all pages.
- **Contact Form:** Functional contact form with validation (Zod) and API route.
- **Service Pages:** Dedicated pages for each service with detailed information.
- **Industry Pages:** Specialized pages for different industry sectors.
